# Not So Simply Optimized

A fork of [Simply Optimized](https://modrinth.com/modpack/sop). It adds a few utility and visual mods:

* [Mod Menu](https://modrinth.com/mod/modmenu)
* [Carpet](https://github.com/gnembon/fabric-carpet)
* [Iris Shaders](https://modrinth.com/mod/iris)
* [Continuity](https://modrinth.com/mod/continuity)
* [Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat)
* [Voice Chat Integration](https://modrinth.com/mod/voice-chat-interaction)
* [Zoomify](https://modrinth.com/mod/zoomify)
* [Textile Backup](https://modrinth.com/mod/textile_backup)
* [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin)
* [YetAnotherConfigLib](https://modrinth.com/mod/yacl)
